# Introduction
Unified Trade 提供了一个统一的 CeDeFi（中心化+去中心化金融）交易抽象层，集成了账户抽象、交易执行、策略编排和跨平台连接。
它使开发者能够通过标准 API 管理完整的交易生命周期——账户创建、交易信号、执行和生态系统集成，消除了连接交易所和 DeFi 协议的复杂性。
它将碎片化的 CeFi 与 DeFi 交易环境统一为一个可编程、可组合、可扩展的执行平面，使开发者专注于策略与产品创新，而非基础设施对接。

为开发者带来的价值
极大降低集成成本：无需分别对接 Binance、OKX、Uniswap、Perp DEX 等系统
提升系统可扩展性：策略与执行后端解耦，支持快速新增交易场所
支持高级交易场景：CeDeFi 组合策略、跨市场对冲、自动化资产管理
为上层应用提供基础设施：钱包、交易终端、策略平台、AI Trader 均可复用

核心能力拆解
1. 统一账户与身份抽象

通过 Crypto Passport / AA 实现 CeFi 账户与 DeFi 钱包的统一表示

支持多交易所账户、多链地址、多资产托管模型

账户权限、签名与执行策略解耦，便于自动化与合规扩展

2. 统一交易执行模型

抽象 CEX API 与 DeFi 合约调用为统一的 Trade Intent / Trade Call

支持现货、永续合约、结构化产品等多类型交易

可在不同执行后端（CEX、Perp DEX、AMM、RWA 协议）之间无缝切换

3. 策略与执行解耦（Strategy Orchestration）

将「信号生成」「风控规则」「执行路径」模块化

支持量化策略、自动对冲、跨平台套利、收益再平衡

可与 AI Agent、外部策略引擎或第三方信号源集成

4. 跨平台与跨生态连接

无需开发者自行处理：

多交易所 API 差异

资金桥接与流动性迁移

CeFi ↔ DeFi 的执行语义差异

通过 Unified Trade API 实现“一次集成，多端执行”
